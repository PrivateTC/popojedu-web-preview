Pravidla (zlatá)

  • Jedna modrá na sekci. Plná #1E88B5 patří jen hlavní akci dané sekce. Vše ostatní je tmavé/neutrální.
  • Modrá = konverze, neutrál = ovládání. Jazyk, měna, Menu, šipky, filtry nikdy nejsou modré.
  • Šipka „→" jen na primáru. Sekundár a ovládání bez šipky.
  • Pryč se starou #58C2F4. Primár vždy #1E88B5.
  • Pohyb střídmě. Glow puls max 1× na stránce; vždy respektovat „reduced motion".

Implementace — CSS základ pro Martina

Kanonické definice (níže). Existující třídy se na ně namapují přes named-scope — bez přepisu markupu, aby se nerozbil booking flow. Globálně nahradit #58C2F4 → #1E88B5 u primárů; sekundáry na ink/bílý ghost.

:root{
  --ptc-blue:#1E88B5;  --ptc-blue-dark:#176f93;
  --ptc-ink:#1C1C2E;
  --ptc-nfill:#EEF1F4; --ptc-nline:#D3D9E0; --ptc-nink:#374151;
}
/* ===== PRIMÁR ===== */
.ptcz-book-cta, .ptcz-bar-primary, .newbuttono.is-primary,
.newbuttonp--primary, button.newbutton[type="submit"]{
  background:var(--ptc-blue); color:#fff; font-weight:700;
  border:2px solid transparent; border-radius:100px;
  box-shadow:0 4px 14px rgb(30 136 181/.35);
  transition:background .18s, box-shadow .25s, transform .18s;
}
.ptcz-book-cta:hover, .ptcz-bar-primary:hover{
  background:var(--ptc-blue-dark); transform:translateY(-1px);
  box-shadow:0 6px 20px rgb(30 136 181/.5);
}
/* ===== SEKUNDÁR — ink ghost (světlé pozadí) ===== */
.naplanovat-btn, .ptcz-bar-outline, .newbuttonp--secondary,
.newbuttono:not(.is-primary){
  background:transparent; color:var(--ptc-ink);
  border:2px solid var(--ptc-ink); border-radius:100px;
  font-weight:700; text-decoration:none;
}
.naplanovat-btn:hover, .ptcz-bar-outline:hover{ background:var(--ptc-ink); color:#fff; }
/* ===== SEKUNDÁR na fotce (hero) ===== */
.ptcz-hero .btn--secondary{ color:#fff; border-color:#fff; }
.ptcz-hero .btn--secondary:hover{ background:#fff; color:var(--ptc-ink); }
/* ===== TERCIÁR — text ===== */
.newbuttonp{ background:none!important; border:none!important; color:var(--ptc-blue);
  font-weight:600; text-decoration:none; }
.newbuttonp:hover{ text-decoration:underline; }
/* ===== NEUTRÁL — ovládání (jazyk, měna, menu, šipky) ===== */
.header_curr, .lang-switch, .ptcz-mm__toggle, .carousel-arrow{
  background:#fff; color:var(--ptc-nink);
  border:1px solid var(--ptc-nline); border-radius:10px; font-weight:500;
}
/* ===== ANIMACE ===== */
@keyframes ptcGlow{0%,100%{box-shadow:0 4px 14px rgb(30 136 181/.30)}
  50%{box-shadow:0 6px 26px rgb(30 136 181/.70)}}
.ptcz-book-cta{ animation:ptcGlow 2.4s ease-in-out infinite; } /* glow jen na sticky */
@media (prefers-reduced-motion: reduce){
  .ptcz-book-cta{ animation:none } *{ transition:none!important }
}
