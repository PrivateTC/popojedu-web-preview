Rozcestník — nový web Popojedu

Všechny verze předlohy, návrhy a dokumenty na jednom místě. U návrhů funguje Režim komentářů (tlačítko vpravo nahoře na každé stránce): klikáte na konkrétní místa, píšete poznámky, „Zkopírovat komentáře" je vyexportuje jako text.

Jak se verze řídí: podle verze v1-2 se právě staví — je závazná a nemění se. v1-3 je připravovaná další etapa (do stavby se zatím nezapracovává). Po nasazení a odsouhlasení v1-2 se v1-3 stane platnou předlohou etapy 2.

1 · Web pro zákazníky (předloha)

Předloha v1-2AKTUÁLNÍ — podle ní se staví

Závazný vzhled i chování nového webu: homepage s konfigurátorem, vozy, ceník, poptávka, kontakt. Jediný zdroj pravdy pro současnou stavbu.

Předloha v1-3 — účet zákazníkaNÁVRH — připravovaná etapa

Celý web v1-2 + nová sekce „Můj účet": rezervace se stavovým pásem (schválení → smlouva → záloha → převzetí → vyúčtování), QR platby, smlouvy, protokoly a doklady. Otevřete menu → Můj účet; stavy přepínáte tmavou pilulkou DEMO vpravo dole.

Předloha v1-1ARCHIV

Předchozí verze — jen pro porovnání historie, nepoužívat.

2 · Administrace (pro provoz)

Admin — rezervační procesNÁVRH — připravovaná etapa

6 obrazovek: rozšířený detail rezervace (platby, kauce, smlouva, doklady, vyúčtování, pokuty), potvrzování s návrhem náhradních vozů, kalendář flotily, fronta plateb, dashboard „Dnešek" a mobilní předávací/vratkový protokol s fotoprůvodcem.

Živý admin (new.popojedu.cz/admin)VÝVOJOVÉ PROSTŘEDÍ

Skutečná administrace rozestavěného webu (Filament) — vyžaduje přihlášení. Zatím obsahuje obsah, poptávky, rezervace, vozy a ceny; obrazovky z návrhu výše přibudou v etapě 2.

Rozestavěný web (new.popojedu.cz)VÝVOJOVÉ PROSTŘEDÍ

Aktuální stav Martinovy stavby podle v1-2 — tady se kontroluje, jak realizace odpovídá předloze.
